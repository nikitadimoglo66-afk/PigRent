# PigRent Plugin

**PigRent** - это плагин для аренды свиней на Minecraft сервере Paper 1.21.1!

## Возможности

✨ **Аренда свиней**
- 30 сек за 1000$
- 1 мин за 2000$
- 5 мин за 5000$

🐷 **Свинья следует за игроком** везде, куда он пойдёт

⚠️ **Предупреждение** за 10 сек до окончания аренды

🏍️ **Зона парковки** - выделенная область, где нужно припаркать свинью

💰 **Интеграция с Vault и EssentialsX** для системы экономики

## Установка

1. Скачивай JAR файл плагина
2. Положи в папку `plugins/`
3. Перезагрузи сервер
4. Установи зону парков��и командой: `/setpark x1 y1 z1 x2 y2 z2`

## Команды

- `/rent short` - Аренда на 30 сек (1000$)
- `/rent medium` - Аренда на 1 мин (2000$)
- `/rent long` - Аренда на 5 мин (5000$)
- `/parkpig` - Показать зону парковки
- `/setpark <x1> <y1> <z1> <x2> <y2> <z2>` - Установить зону парковки (админ)

## Компиляция

```bash
mvn clean package
```

Готовый JAR файл будет в папке `target/`

## Требования

- Java 21+
- Paper 1.21.1
- Vault
- EssentialsX
- Maven (для компиляции)
